## 🎓 Lessons Learned
 
- **Automation is mandatory** — manual solving is impossible for speed-based challenges
- **RSA requires math depth** — breaking it needs number theory, not just coding
- **DevTools reveals secrets** — session cookies and headers can leak critical info
- **Time management matters** — incomplete scripts cost flags under pressure
- **Efficiency over brute force** — CPU-based attacks need optimized, not naive, code
